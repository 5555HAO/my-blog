html的基础框架

  • 📅2024年7月20日

  • 💻编程

html是什么？

HTML5是用来描述网页的一种语言，被称为超文本标记语言。用HTML5编写的文件，后缀以 .html 结尾，HTML是一种标记
语言，标记语言是一套标记标签。标签是由尖括号包围的关键字，例如： < html >（没有空格） 标签有两种表现形式：

  1. 双标签,例如:< html > < /html >

  2. 单标签,例如:< img >

html的基础框架是什么?

在任意一个盘的路径下创建一个名为 Welcome.html 的文件，并在文件中写入以下代码：

                        < html >   —— 创建网页时必定的写入的代码（专属代码）
                            < head >   —— 浏览器头部信息 （网页名）
                                < title >我的网页< /title >   —— 网页标题
                            < /head >
                            < body >   —— 网页中显示内容
                                Hello,我的第一个网页
                            < /body >
                        < /html >   —— 在其中写完代码后就用一个斜杠然后…表示结束

html5中的!DOCTYPE声明

DOCTYPE 是 document type (文档类型) 的缩写。 是H5的声明，位于文档的最前面，处于标签之前。它是网页必备的组成
部分，避免浏览器的怪异模式。 H5网页必备的声明（第一行代码）：
< !DOCTYPE html >

HTML5的基本骨架

每个网页都会有的基本骨架：
< html >< /html >
< head >< /head >
< body >< /body >


释义：

< html > 标签（双标签）： 定义 HTML 文档，这个元素我们浏览器看到后就明白这是个 HTML 文档了，所以你的其它元素
都要包裹在它里面，标签限定了文档的开始点和结束点：
< !DOCTYPE html >
< html >
< /html >


< head > 标签（双标签）： head 标签用于定义文档的头部。文档的头部描述了文档的各种属性和信息，包括文档的标题、
在 Web中的位置以及其他文档的关系等。绝大多数文档头部包含的数据都不会真正作为内容显示给读者：
< !DOCTYPE html >
< html >
          < head >
          < /head >
< /html >


< body > 标签（双标签）：
< body > 元素定义文档的主体
< body > 元素包含文档的所有内容（比如文本、超链接、图像、表格和列表等等）
它会直接在页面中显示出来，也就是用户可以直观看到的内容：
< !DOCTYPE html >
< html >
          < head >
          < /head >
          < body >
                        我会显示在浏览器中
          < /body >
< /html >

title与meta标签

< title >标签（双标签）：
可定义文档的标题
它显示在浏览器窗口的标题栏或状态栏上
< title > 标签是 < head > 标签中唯一必须要求包含的东西，就是说写 < head > 一定要写< title >
< title >的增加有利于SEO优化（SEO 是搜索引擎优化的英文缩写，通过对网站内容调整，满足搜索引擎的排名需求）：
< !DOCTYPE html >
< html >
          < head >
                    < title >我是一个标题< /title >
          < /head >
          < body >
                     我会显示在浏览器上
          < /body >
< /html >


< meta > 标签（单标签）：
< meta > 标签是用来描述一个 HTML 网页文档的属性，关键词等，例如：charset=”utf-8”
是说当前使用的是 utf-8的编码格式，在开发中我们经常会看到 utf-8 , 或是 gbk , 这些都是编码格式，通常使用 utf-8 格式：
< !DOCTYPE html >
< html >
          < head >
                    < title >标题< /title >
                    < meta charset=”utf-8” >
          < /head >
          < body >
                    你好
          < /body >
< /html >
